## 📖 Project Overview

This project performs a complete **Exploratory Data Analysis (EDA)** on the **Sample Superstore Dataset** using Python.

The project follows the complete Data Analytics workflow:

- 📥 Data Loading
- 🔍 Data Understanding
- 🧹 Data Cleaning
- 📊 Exploratory Data Analysis (EDA)
- ⚙️ Feature Engineering

The objective is to transform raw business data into meaningful insights that support data-driven decision-making.
